02

¿Qué es un formulario?

Un formulario permite que el usuario ingrese información dentro de una página web.

📝

Captura datos

Permite pedir información como nombre, edad, correo, grupo, teléfono o comentarios.

✅

Valida información

Revisa que los campos no estén vacíos y que el usuario escriba datos correctos.

⚡

Usa JavaScript

JavaScript puede leer lo que escribió el usuario y mostrarlo dentro de la página.

Estructura básica de un formulario

<form>
  <label for="nombre">Nombre:</label>
  <input type="text" id="nombre">

  <button type="button">Enviar</button>
</form>
03

Etiquetas principales

Estas etiquetas son la base para construir formularios en HTML5.

<form>

Contenedor principal del formulario.

<label>

Indica qué dato debe escribir el usuario.

<input>

Crea campos para escribir texto, números, correo o contraseña.

<textarea>

Permite escribir comentarios o textos largos.

<select>

Crea una lista desplegable de opciones.

<button>

Ejecuta una acción al presionarlo.

04

Tipos de input

HTML5 incluye varios tipos de campos según el dato que se necesita.

Tipo Uso Ejemplo
text Texto normal <input type="text">
number Números <input type="number">
email Correo electrónico <input type="email">
password Contraseña <input type="password">
date Fecha <input type="date">

06

Código del demo

Aquí pueden ver el código del ejemplo básico para realizarlo en su laptop. Selecciona HTML, CSS o JavaScript. 

<section class="demo-form-card">
  <h3>Formulario básico</h3>
  <p>Completa los campos y presiona el botón.</p>

  <form>
    <label for="nombre">Nombre:</label>
    <input type="text" id="nombre" placeholder="Escribe tu nombre">

    <label for="edad">Edad:</label>
    <input type="number" id="edad" placeholder="Escribe tu edad">

    <button type="button" onclick="mostrarDatos()">Mostrar datos</button>
  </form>

  <div id="resultado" class="resultado">
    Aquí aparecerá la información capturada.
  </div>
</section>

function mostrarDatos(){
  let nombre = document.getElementById("nombre").value.trim();
  let edad = document.getElementById("edad").value.trim();
  let resultado = document.getElementById("resultado");

  if(nombre === "" || edad === ""){
    resultado.innerHTML = "⚠️ Por favor completa todos los campos.";
    resultado.style.background = "#ffe0d6";
    resultado.style.border = "2px solid #e34f26";
  } else {
    resultado.innerHTML =
      "<strong>✅ Datos capturados correctamente:</strong><br><br>" +
      "Nombre: " + nombre + "<br>" +
      "Edad: " + edad + " años";

    resultado.style.background = "#e8ffe8";
    resultado.style.border = "2px solid #2ecc71";
  }
}
